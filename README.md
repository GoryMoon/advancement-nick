# Advancement Nick

This is a plugin to replace the player name in advancement packets with the [Vault][1] Nick name.

## Setup

To build the plugin run:

```shell script
# Windows
gradlew.bat build
```
```shell script
# Linux
./gradlew build
```

The plugin will be located in `build/libs/` when done building.
